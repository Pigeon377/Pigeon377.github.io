关于我...

2022年09月13日

这大概算是自我介绍

下列内容大都来自 2022年福州大学软件工程实践 要求的第一次作业。
有少量删改

另外，第二次作业要求中的上传照片
因为个人长相问题以及隐私需求
容我拒绝

自我介绍

生活相关

我是🐟🐋，来自福州大学2020级数据科学与大数据技术专业。

是一个技术品味相当糟糕的家伙。

兴趣是烹饪和coding(如果够资格的话)，万一年纪大了被优化了，想去当个厨子🤭。

没什么特点的普通学生，成绩中游，不过体重很有可能是超一流。

专业相关

24岁，是学生。

目前是学生，平日里喜欢在github上写点小玩具。

技术力不高，暂时能支持:

  • 单人开发简单的前后端分离应用，
  • 进行部署
  • 具有解决BUG的能力(指读不懂stackoverflow就去找github的issue，还是找不到就重启)

我认为我还缺少:

  • 团队协作的经验和能力
  • 进行调优的能力
  • 积极参与开源社区的特定领域相关知识和勇气
  • 流畅阅读英语资料的能力
  • 进行完整且规范的开发的能力

语言

比较熟悉的是

  • Scala3/Dotty
  • Golang

但是一些比较高级的特性和知识，比如Scala的Marco,
Golang的Slice内存如何分配之类的，了解不多。

因为很喜欢写代码，所以会一些其他语言:

  • java (写Web后端或者小玩具)
  • C (主要是拿来刷题)
  • Python (写Web后端，爬虫以及ML)
  • JavaScript (搓前端的页面交互逻辑，以及和后端的通信)

当前正在学Rust，希望世界上能少一点所有权,多一点gc。

还有CPP 能不能赶紧退环境啊

框架

  • 前端
    • html
    • css
    • javascript
    • Vue
  • 后端
    • Flask
    • Gin
    • Akka
    • SpringBoot

前端除了三件套，还会一点Vue，只是勉强能用的水平，布局管理等水平极其低下。通常依赖Element-Plus组件来进行开发。

后端比较熟悉Flask及其相关组件(像是Flask-SQLAlchemy之类的第三方扩展)，除此之外，还有Gin、Akka、SpringBoot等框架的使用经验，但谈不上很熟悉，也只是能够正常使用常见功能，完成CRUD业务的开发而已

技术取向

  • 数据库
  • 分布式
  • 一点点ML
  • 没了

个人对数据库方面的知识非常感兴趣，目前在学习CMU 15-445和MIT 6.824两门课程，但是都只做了几个Lab。之前想试着实现简单的内存型数据库和磁盘型数据库，写完之后发现惨不忍睹，transcation、join algorithm、buffer pool之类的组件完全写不出来，没有足够的理论知识支撑，所以现在在努力学习基础知识😥。

更新:

CMU 15-445的lab用的是C++，这语言懂得都懂，五年开发经验都不敢说自己很熟，我实在是不太啃得动。
实在是顶不住了，现在转头去写UCB CS186的lab了，刚刚搓到Project 3: Joins and Query Optimization

因为时代背景，也在试着学习ML，但是兴趣不如数据库强烈。

未来规划

打算试着先投下实习，如果就业仍然很困难，会试着考研。

继续学习，争取这个学期把几门网课的课程和Lab啃完。
同时提高自己的工程素养。

大三下先投递实习，希望能投中 Golang后端开发/存储的岗位。

会着手准备考研，但是优先工作。

希望能够顺利找到工作或上岸吧。

等我打完了这份工 就回老家开家沙县小吃😭😭😭
