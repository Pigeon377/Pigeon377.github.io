关于导入了maven依赖却仍然没办法用这件事

2021年12月22日

RT

我确实是想不到一天能因为技术力低下水两篇小blog…

…真的给自己整无语了。

起因是书上说要导入akka-testkit依赖测试actor,结果从去maven仓库拉了一下,拉下来居然找不到这个包…

重点来了…今天翻怎么将slf4j绑到log4j上面时,发现..特么的爷依赖的<scope>是test…我直接就无语了好吧。

顺手记一下:

scope 作用域 依赖传递
compile all 是
runtime runtime,test 是
provided compile, test 否
test test 否
system compile, test 是
